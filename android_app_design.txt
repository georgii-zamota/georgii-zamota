Навигация

1. Кнопка назад отрабатывает корректно
2. Жест для возвращения на главный экран отрабатывает корректно
3. Жест для перемещения между экранами отрабатывает корректно
4. Анимация пермещения между экранами отрабатывает корректно
5. Приложение сохраняет данные при сворачивании и выходе из приложения на главный экран и возвращение обратно
6. Приложение сохраняет данные при сворачивании и выходе из приложения  другое приложение и возвращение обратно
7. Приложение сохраняет данные при сворачивании и выходе из приложения после заблокировки, разблокировке телефона и возращения обратно в приложение
8. Приложение сохраняет данные при заблокировки экране и разблокировки
9. Приложение сохраняет данные при сворачивании и выходе из приложения на некоторое время (в минутах) и возвращение обратно

Уведомления

10. Уведомления не используются для продвижения или рекламы другого продукта 
11. Уведомления с приложения приходят
12. Уведомления с приложения не приходят, если их отключить
13. Уведомления собираются в одну группу уведомлений
14. Установить тайм-ауты для работы уведомлений, там где это возможно
15. Уведомления сохраняются, только если они привязаны к текущим событиям (телефонный звонок, воспроизведение музыки)
16. Уведомления СМС поддерживают действия прямого ответа
17. Уведомления ответа на сообщения из мессенджеров поддерживают действия прямого ответа

Интерфейс и графика

18. Приложение поддерживает альбомную ориентацию, если это возможно 
19. Приложение поддерживает портретную ориентацию, если это возможно
20. Приложение поддерживает его сворачивание в режим ожидания и возвращение обратно
21. Приложение корректно отрабатывает переход из альбомной ориентации в портретную
22. Приложение корректно отрабатывает переход из потретной ориентации в альбомную

Качество визуала

23. Приложение корректно отображает графику
24. Приложение корректно отображает цвета
25. Приложение корректно отображает текст
26. Приложение корректно отображает изображения
27. Приложение использует векторную графику там, где это возможно
28. Приложение корректно обеспечивает воспроизведение графики для всех целевых вариантов разрешения экранов
29. Кнопки навигации отображаются корректно
30. Перенос слов внутри кнопок отрабатывает коректно
31. Расстояния между буквами и словами текста достаточно
